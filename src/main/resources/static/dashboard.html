Resumo do Sistema

O AgroScan IoT utiliza ESP32, sensores ambientais e uma API REST para monitorar diferentes categorias agrícolas. O sistema realiza o escaneamento da amostra, classifica entre Planta, Verdura, Fruta, Legume ou Solo e apresenta o status em tempo real.

Atualização automática a cada 3 segundos
